The Studio

analog signal path · hardware synthesis · no plugins

Hardware Synths → Patchbay → SSL SiX → Main Out → Focusrite Clarett+ 8Pre (inputs 1/2) → Computer
Hardware Synths → Patchbay → Focusrite Clarett+ 8Pre (inputs 3-8) → Computer → Web

𒌓 Mixing & Interface

SSL SiX

Analog Mixer

Solid State Logic desktop mixer. Two SuperAnalogue channel strips with EQ and dynamics. Main outs routed to Clarett inputs 1/2. The center of the signal path.

Focusrite Clarett+ 8Pre

Audio Interface

8 analog inputs, Thunderbolt/USB-C. Ins and outs fully routed through the patchbay. The bridge between hardware and computer — and now, the web.

Patchbay

Signal Routing

All audio and MIDI routed through a centralized patchbay. Normalled for standard chains, repatchable on the fly. No cable crawling.

𒀭 Synthesizers

Roland Juno DS

Polysynth · Workstation

Versatile polysynth with deep preset library. Pads, leads, electric pianos. Routed to SSL SiX / Clarett inputs 3/4.

MIDI Port 4

Korg Minilogue

4-Voice Analog Polysynth

True analog oscillators, 4 voices. The hands-on synth — every knob is a parameter. Warm, immediate, alive.

MIDI Port 7

Behringer VC340

String Machine · Vocoder

Clone of the Roland VP-330. Lush analog strings and choir, built-in vocoder. Routed to Clarett inputs 5/6.

MIDI Port 5

Akai Timbre Wolf

4-Voice Analog Polysynth

Four independent analog voices with built-in sequencer. Raw, lo-fi character. Each voice tunable independently — good for detuned unison or 4-part sequences.

Roland JV-1010

Sound Module

Half-rack sound module. Massive library of 90s/2000s sounds — orchestral, pads, world instruments. Routed to Clarett inputs 7/8.

MIDI Port 3

𒁹 Drum Machines

Behringer RD-8

Analog Drum Machine · 808 Clone

Analog recreation of the TR-808. Kicks that move air. Stereo output through patchbay.

MIDI Port 2

Alesis SR-16

Drum Machine

Classic digital drum machine. 233 sounds, swing control, multiple outputs. A workhorse.

MIDI Port 6

Roland R8

Drum Machine

90s flagship drum machine. Expressive, velocity-sensitive pads. Deep sound library with expansion cards. A classic.

MIDI Port 1

𒈗 Effects

Yamaha R1000

Digital Reverb

80s rack reverb. Warm, distinctive character. Routed through patchbay for any source.

Fostex 3510

Spring Reverb

Hardware spring reverb unit. The kind of character that no plugin replicates. Physical springs, real resonance.
