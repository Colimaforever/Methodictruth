📖 Synth Guide

Modules

Oscillator
sine = pure/smooth · saw = bright/buzzy · square = hollow/retro · triangle = soft/mellow
Filter
LP cuts highs (warmth) · HP cuts lows (thin/bright) · BP isolates a band. Cutoff = where it cuts, Resonance = emphasis at cutoff.
LFO
Slow modulation for movement. Rate = speed, Depth = intensity. Route to pitch = vibrato, filter = wah, volume = tremolo.
FM (Frequency Modulation)
Patch one oscillator's out into another's FM input to modulate its pitch at audio rate. FM Amt = depth in Hz of frequency deviation. Set the modulator at a simple ratio of the carrier: 1:1/2:1 = harmonic (clean, organ-like), 3.5:1+ = inharmonic/metallic (bells, e-piano, gongs).
Envelope (ADSR)
Attack = fade-in time · Decay = drop after peak · Sustain = held level · Release = fade after note off. Patch it between an oscillator and Output and it re-fires on every note — piano keys, MIDI, and the sequencer all retrigger it, so a short Decay gives plucked/percussive steps instead of a flat on/off.
Delay
Echo effect. Time = gap between repeats · Feedback = how many repeats · Mix = dry/wet balance.
Reverb
Room simulation. Size = room size · Damping = high-freq absorption · Mix = how much reverb.
Gain
Volume control — use to balance levels between modules.
Noise
white = all freqs equal (hiss) · pink = bass-heavy (wind) · brown = deep rumble.
Audio File
Drop an audio file onto the module (or click Load) to play and loop it — drop in several to build a multi-track mix. Vol sets its level.
Pan
Places a sound left/right in the stereo field. −1 = full left, 1 = full right.
EQ
3-band tone shaping. Cut Low to thin out a track, boost High for clarity, pull Mid down to make room for another track ("carving").
Compressor
Evens out volume swings. Thresh = level it kicks in at · Ratio = how hard it squeezes · Atk/Rel = how fast it reacts/lets go.
Chorus
Layers slightly-detuned copies for a thicker, shimmering sound. Rate/Depth = how the detune wobbles.
Limiter
A hard ceiling on loudness — put one right before Output to stop a mix from clipping. Thresh = the ceiling.

Module Categories

Every toolbar button and module card is color-coded by its role in the rack, so a glance tells you what a patch is made of:

  • Source — Oscillator, Noise, Audio File: where the sound starts.
  • Shape — Filter, EQ, Envelope: where its tone and contour are carved.
  • Mod — LFO: slow movement patched into another module's parameter.
  • Mix — Gain, Pan, Compressor, Limiter: levels and stereo placement.
  • Space FX — Delay, Reverb, Chorus: depth and width.
  • Out — Output: the final stage, with a live VU meter showing how hot the signal is running.

Mixing Multiple Tracks

Build a mix
Add one Audio File module per stem, route each through its own Pan/EQ into a shared Gain or Compressor, then into one Output. Add a Limiter just before Output to keep the combined level in check.
+ Channel Strip
One click drops a pre-cabled Pan → EQ → Compressor → Limiter → Output chain — the standard mixing order — so you can patch a source straight into a finished signal chain instead of wiring five modules by hand.
VU Meter
Every Output module shows a live level bar under its Master slider — green while there's headroom, amber as it gets hot, red near clipping. Pull the Master fader or add a Limiter upstream if it's pinned red.

Sequencer

Step Sequencer
Open with 🎼 Sequencer in the toolbar. It plays your patched sound — build a basic synth first (Oscillator → … → Output), press Start Audio, then hit Play. Tap a step to turn it on/off; use the −/+ under an active step to change its pitch. Set length (8/16/32 steps), BPM, swing, and gate length. The matching piano key lights up as each step fires. Patterns auto-save and reload.
Shaping each step (Decay/ADSR)
The sequencer has no envelope of its own — it plays through whatever's patched on the canvas. Add an ADSR module before Output and set a short Dec (decay) with low Sus (sustain) for plucky, percussive steps, or a longer Decay/higher Sustain for pads that bleed into the next step. It retriggers on every step automatically.

MIDI — Controllers and DAWs

Play from a MIDI controller
Plug in a USB MIDI keyboard and allow MIDI access when Chrome prompts you. The MIDI: Connected badge confirms it. CC1 (mod wheel) sweeps filter cutoff; CC74 sweeps resonance, on every Filter module in the patch.
Send notes out to Logic Pro X
The MIDI Out dropdown sends every note this page plays — piano keys, MIDI controller passthrough, and the step sequencer — to an external MIDI port in real time, so Logic Pro X can record or play them through one of its own instruments. Requires three things: (1) Chrome or Edge — Web MIDI isn't supported in Safari or Firefox. (2) A virtual MIDI port — on macOS, open Audio MIDI Setup → Window → Show MIDI Studio, double-click IAC Driver, check Device is online, and make sure it has at least one port (the default "Bus 1" is fine). (3) A Logic Pro track listening on that port — create or select a Software Instrument track, open the track's input in the inspector, and set it to receive from the IAC bus you just enabled (or "All Inputs"). Once that's done, pick the IAC bus from the MIDI Out dropdown here — notes play through your local patch and get sent to Logic at the same time.

Patching Tips

  • Basic synth: Oscillator → Filter → Gain → Output
  • Add movement: LFO → Filter cutoff
  • Add space: Output → Delay → Reverb
  • Thick sound: Two oscillators slightly detuned

Sound Design Recipes

  • Warm pad: Saw osc → LP filter (low cutoff, med resonance) → slow LFO on filter → long attack/release
  • Bass: Square osc → LP filter → short attack, med sustain
  • Lead: Saw osc → BP filter (high resonance) → short attack, med release
