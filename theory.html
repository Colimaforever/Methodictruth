Maqam & Middle Eastern Scales
▾

Maqam (Arabic: مقام) is a system of melodic modes used in Arabic, Turkish, and Azerbaijani music. Unlike Western scales which define a static set of notes, maqam emphasizes melodic movement — specific patterns of ascent and descent, characteristic phrases, and microtonal inflections that give each maqam its distinctive emotional character.

Mugham (Azerbaijani), Makam (Turkish), Maqam (Arabic) — different traditions, shared roots.

⚠ Some maqam scales use quarter tones (halfway between Western semitones). The versions here are approximated to 12-tone equal temperament for piano and guitar. True maqam performance uses microtonal inflections that can't be fully captured on fretted instruments.

Modes Reference
▾

Modes are the same set of notes as the parent major scale, but starting from a different degree. Each mode has its own unique intervallic character and emotional quality.
