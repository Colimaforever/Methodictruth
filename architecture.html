🌿 Key Concepts 🌿
1 Messages arrive via multiple channels but converge into a single session. Telegram, webchat, Discord — they all become the same format by the time the agent sees them.
2 The agent doesn't remember between sessions — memory files are its continuity. SOUL.md is loaded fresh every time; MEMORY.md is curated long-term recall.
3 Sub-agents run in isolated sessions for parallel work. Need to build a page while answering a question? Spawn a sub-agent. This very page was built by one.
4 Tools give the agent hands — file access, web search, code execution, browser control. Without tools, it's just a chatbot. With them, it's an operator.
5 The personality (SOUL.md) is loaded every session — identity is a file. Change the file, change who the agent is. Truth's identity persists because the file persists.
